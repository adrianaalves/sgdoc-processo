Descrição

Cadastrar dados de trâmite

 

Esta opção possibilita que o usuário registre o encaminhamento de documentos físicos para outro órgão. Observar a aba em destaque para realizar a ação desejada.

 

 

CAMPO

O QUE INSERIR

Documento

Número da digital do documento

Destinatário

Nome ou cargo da pessoa que receberá o documento

Local

Estado, empresa ou órgão de destino do documento

Endereço

Endereço do órgão ou empresa

Telefone

Número para contato com o destinatário

CEP

Código postal do estado, empresa ou órgão de destino

Prioridade

Selecionar a prioridade do documento

 

Nota: Para realizar um tramite externo todos os campos apresentados na tela acima, deverão ser preenchidos. Após o preenchimento dos campos, para confirmar o tramite clique no botão

 

Todo o procedimento realizado no sistema lhe apresentará a seguinte mensagem:

 

 

Ao Confirmar clicando no botão ok irá mostrar uma tela com as informações do tramite, apresentando a Guia de Encaminhamento de Documento pelo o sistema SGDOC.

 

A Guia só deve ser impressa se o documento for ser entregue pessoalmente em outro órgão. Ao tramitar o documento o mesmo irá para a caixa de externos podendo ser resgatado para a área de trabalho.
